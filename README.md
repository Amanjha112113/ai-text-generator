## Quick Start
1. Clone: `https://github.com/Amanjha112113/ai-text-generator.git`
2. Env: Copy `.env.example` to `.env`, add `GEMINI_API_KEY=your_api_key`.
3. Install: `pip install -r requirements.txt`
4. Run: `streamlit run app.py`
